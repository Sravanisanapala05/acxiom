# ACXIOM Library Management System

## Introduction
ACXIOM Library Management System is an online application built using Python and Gradio. The project manages library management activities like issuance of books, returning of books, availability checking, and fines.

Hugging Face Spaces technology is used to deploy this application online.

## Features
- User and Admin Login
- Check availability of a book
- Issuance and Returning of books
- Automatic generation of return date after issue (15 days)
- Calculation of fine after delay in returning the book
- Verification of fines before returning
- Autocompletion of author name according to the selected book
- Books, Issued Books Report and Pending Fines Report

## Technologies Used
- Python
- Gradio (Frontend)
- Pandas (For Data Manipulation)
- Hugging Face Spaces (Deployment)

## Working
- Log in to the system
- Check book availability
- Book will be issued
- Return date will be generated automatically by the system
- Fines will be calculated for delayed returning
- Pay fines for returning a book
- Generate reports for all operations
